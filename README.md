# numeric_dependency_analysis

## Цель проекта
Исследовать зависимости между числовыми данными и проверить, как на вывод влияют шум, малое количество данных и проверка на новых точках.

## Задачи
1. Сформировать идеальные числовые данные для простой функции (например, y = x^2)
2. Добавить шум и проверить, как меняется вывод
3. Уменьшить количество точек и оценить доверие к гипотезе
4. Разделить данные на обучающую и тестовую часть (train/test split) и проверить предсказания

## Технологии
- Python 3
- Matplotlib (для графиков)
- NumPy (для вычислений)
